# Bioinformatics RNA-Seq Analysis of Aedes aegypti

## Overview
This project investigates differential gene expression associated with temephos resistance in Aedes aegypti using RNA-Seq data.

## Objectives
- Identify differentially expressed genes (DEGs)
- Compare resistant and susceptible populations
- Visualize transcriptomic changes using volcano plots and heatmaps
- Annotate candidate genes using VectorBase

## Tools and Databases
- R / RStudio
- VectorBase
- Microsoft Excel

## Key Results

- AAEL013757
